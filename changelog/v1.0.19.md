# v1.0.19 - 2026-06-17

## Changed

- `MySQLConfig` adds `database` from Walle `mysql_config.database` (group-level `mysql_db_name`).
- `MySQLDSN` reads database from config (default `game` when empty); removed second `database` argument.

## Added

- `ParseGroupName` validates a single group name (rejects comma-separated values).
- `Client.GetGameGroup` fetches one game group config in a single call.
- `infra/listen` package: `IsLocal`, `Normalize` for HTTP/gRPC listen addresses, `PprofListenAddr`, and port-bump helpers for local dev.
