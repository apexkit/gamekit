# v1.0.14 - 2026-06-17

## Added

- add `AppGroupInfoCache.EvictByGroupID(groupID, id uint64)` to remove cached app group info by `group_id`, and by primary key `id` when `id > 0`

## Usage

Use after disabling or deleting an app group so runtime services do not keep serving stale group config from Redis:

```go
cache.NewAppGroupInfoCache(rdb, db).EvictByGroupID(appGroup.GroupId, appGroup.Id)
```

For sub-merchant (`app_info`) changes, refresh the parent group cache with `RefreshByAppGroupID` instead of evicting per-app cache entries.
