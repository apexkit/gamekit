# v1.0.20 - 2026-06-24

## Added

- `conf` package: shared `Bootstrap` proto/config (Server, Data, Log, Registry, Prometheus, Rpc, Aws), Walle overlay, listen normalization, startup target logging.
- `infra/bootstrap` package: functional options (`WithServiceName`, `WithServiceID`, `WithVersion`, `WithConfPath`, `WithWalle`, `WithPprof`, `WithMysql`, `WithRedis`), `NewRuntime`, and wire `ProviderSet`.
- `infra/store/mysql` and `infra/store/redis`: atreus-style `NewOptions` + `NewDBManager` / `NewClient` from shared `conf.Data`.
- `provider.StoreProviderSet`: central wire set for mysql/redis stores.
- `log.NewZapLoggerForEnv`: production JSON logger or local dev console logger based on `IS_LOCAL`.

## Changed

- `walle` MySQL/Redis/Consul endpoint resolution uses `IS_LOCAL` (external when local, internal otherwise).
