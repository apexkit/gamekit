# v1.0.23 - 2026-06-29

## Added

- `infra/resource`: Cola-style `Install` — declare `WithMysql` / `WithRedis` / `WithNats` in main, dial at `NewRuntime`, hold clients on `Runtime.Resources`.
- `conf.Data.Eventbus` proto field; Walle `nats_config` on `/openapi/game/group` with `NatsEndpoint` resolver.
- `ApplyByWalle` overlays `data.eventbus` from Walle `nats_config`.
- `bootstrap.ProvideRegistrar`, `ProvideDatabases`, `ProvideRedisClient`, `ProvideEventBus`.
- `eventbus.NewConnectedNatsBus` for startup connectivity check.

## Changed

- `NewRuntime` calls `resource.Install`; wire reads clients via `Provide*` from `Runtime.Resources` (no per-service `StoreProviderSet` / `EventbusProviderSet` required).
- Local default listen ports when `addr` is empty: HTTP `8080`, gRPC `9090` (aligned with production; port bump on conflict still applies locally).
- `provider.StoreProviderSet` and `EventbusProviderSet` deprecated in favor of `bootstrap.ProviderSet` + `With*` flags.
